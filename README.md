# 自动亏钱 Agent (Automated Loss-Making Agent) V0.5

这是一个基于大语言模型（LLM）和 LangGraph 的自动化加密货币交易系统。
目标是通过 AI 驱动的策略管理，实现多维度的自动化交易与风险控制。

## 功能特点

- **多 Agent 隔离架构**: 基于 `config_id` 的完全隔离，支持同一交易对运行不同策略（如 BTC/USDT 15m 短线 vs 1h 趋势）
- **高阶指标系统 (v2)**: 精简冗余指标，新增 MACD 动量标注、RSI 顶底背离检测及自适应成品价分布 (Volume Profile)
- **三种交易模式**:
    - `REAL` 实盘模式: 15分钟心跳，自动管理止损止盈及仓位
    - `STRATEGY` 策略模式: 高保真模拟挂单，用于验证交易计划的盈亏比
    - `SPOT_DCA` 定投模式: 现货定投逻辑，支持按日/周灵活配置
- **多维性能分析**: 实时追踪账户净值、成交历史、Token 消耗统计及基于 `pricing.json` 的模型计价
- **智能选择器 (Router v3)**: 引入初筛模型（Small Model），自动识别行情机会，动态决定是否触发深度分析（Master Model），大幅降低运行成本
- **长短期记忆**:
    - 短期：保留近 10 次分析汇总，维持逻辑连贯性
    - 长期：每日自动汇总全天逻辑，生成行情回顾，为 Agent 提供宏观背景
- **可视化面板**: 完整的 Web 仪表盘，支持在线改配、实时监控及 SSE 实时对话审批交易

## 配置说明

### 1. 币安 API 配置
在 `.env` 文件中填入您的 API Key，确保有"读取"和"交易"权限。

### 2. 调试工具 (LangSmith)
如需调试 Agent，可在 `.env` 中设置 `LANGCHAIN_TRACING_V2=true` 启用跟踪。
