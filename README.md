# 自动亏钱 Agent (Automated Loss-Making Agent) V0.3

这是一个基于大语言模型（LLM）和 LangGraph 的自动化加密货币交易系统。
目标是通过"自动亏钱"反向验证策略的有效性。

## 功能特点

- **多 Agent 架构**: 每个交易对可以独立配置LLM、杠杆、提示词和策略参数
- **三种交易模式**:
    - 实盘模式: 15分钟高频扫描，直接进行交易所API操作
    - 策略模式: 1小时模拟挂单，测试高盈亏比的交易计划
    - 定投模式: 现货定投逻辑，自动买入摊低成本
- **可视化配置**: 通过Web界面管理交易对，支持实时开关
- **提示词管理**: 在线编辑策略模板，动态关联到Agent
- **统计监控**: 记录Token消耗，提供多Agent对比视图
- **智能聊天**: 与运行中的Agent对话，人工审批后执行交易

## 配置说明

### 1. 币安 API 配置
在 `.env` 文件中填入您的 API Key，确保有"读取"和"交易"权限。

### 2. 调试工具 (LangSmith)
如需调试 Agent，可在 `.env` 中设置 `LANGCHAIN_TRACING_V2=true` 启用跟踪。
