## <a id="faq"></a>❓ FAQ / 常见问题解答

### Q1: 应该使用什么 LLM 模型？
LLM需要支持function call和OpenAI调用方式。目前来看，越强的模型表现越好。性价比方面，国产的qwen3-max表现不错。

### Q2: 会爆仓吗？
在实盘模式下，15分钟的监控频率，Agent一般不会爆仓，会自动平仓。但遇到极端行情时仍需谨慎，建议配置市值较大的币种进行测试。

### Q3: 建议使用多少仓位进行测试？
建议低杠杆，金额不超过100U进行测试。

### Q4: 如何运行项目，需要服务器吗？
不需要域名或服务器，本地即可运行。如果在大陆使用，可能需要配置代理。

### Q5: 为什么需要清除历史记录？
清除历史记录可以让Agent重新开始分析，避免历史思维影响新的决策。建议每隔几天清理一次。

### Q6: 能稳赚钱吗？
这个项目主要是减少盯盘成本，不能保证稳赚。量化交易始终存在风险。

### Q7: Agent的输入有哪些？
Agent接收技术指标作为输入：RSI，EMA20/50/100/200，资金费率，持仓量，波动率，当前仓位，当前挂单，近3条历史记录等。

### Q8: 实盘Agent的工具有哪些？
支持Buy/Sell限价单操作，CLOSE平仓指令，CANCEL撤销挂单指令。

### Q9: 为什么不推荐deepseek？
DeepSeek输出虽有逻辑性，但实际交易效果不佳，经常出现连续亏损。

### Q10: 需要修改Prompt吗？
是的，Prompt反映了特定的交易思维，需要根据自己的风格和选择的LLM进行调整。

### Q11: 为什么有的Agent输出是英文？
这通常表明所用LLM质量不够好。

### Q12: LLM API 成本是多少？
实盘模式下，使用qwen3-max一天约4元人民币，使用gpt5.2一天约9元人民币。
