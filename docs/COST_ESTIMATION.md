# 成本估算

运行本交易 Agent 的主要成本来自 LLM 的 API 调用。

## 运行成本分析

单次运行的 Token 消耗通常在 3000-4000 Tokens 之间。

| 模式 | 运行频率 | 每日调用次数 | 预估单次 Token | 每日总 Token |
| :--- | :--- | :--- | :--- | :--- |
| 模拟模式 | 1 小时 / 次 | 24 次 | ~3500 | ~84000 |
| 实盘模式 | 15 分钟 / 次 | 96 次 | ~3500 | ~336000 |

### 费用参考 (以 GPT-4o-mini 为例)

- **模拟模式**: 约 $0.02 / 天 / 交易对
- **实盘模式**: 约 $0.10 / 天 / 交易对

> 注意: 如果开启了 Summarizer 或频繁与机器人对话，成本会相应增加。
